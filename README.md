# 📊 IndiaBiz Analytics  
### Sales Forecasting & Revenue Optimization Platform

IndiaBiz Analytics is a full-stack web application that helps businesses analyze sales data, forecast future demand, and optimize revenue using data-driven insights. It provides real-time dashboards, interactive visualizations, and state-wise analytics across India.

---

## 🚀 Features

- 📈 **Sales Forecasting**
  - Predict future sales using moving average
  - Forecast next 30 days
  - Compare actual vs predicted values
  - Metrics: MAPE, MAE, RMSE

- 💰 **Revenue Optimization**
  - Suggest optimal pricing strategies
  - Identify high and low performing products
  - Improve overall revenue

- 🗺️ **India Map Analytics**
  - State and district-level insights
  - Interactive drill-down visualization
  - Region-wise performance tracking

- 📦 **Product & Sales Management**
  - Add and manage products
  - Upload product images
  - Record daily sales transactions
  - Track revenue and stock

- 👥 **User & Admin System**
  - Secure authentication (JWT)
  - Role-based dashboards (Admin & User)

- 📊 **Dashboard & Visualization**
  - Real-time analytics dashboard
  - Interactive charts using Chart.js
  - Modern dark theme UI
